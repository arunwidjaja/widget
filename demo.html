üöÄ Quick Integration

Add the chatbot to your website with just a few lines of code:

<!-- Add this to your HTML head or before closing body tag -->
<script src="dist/widget.js"></script>

<script>
// Initialize the chatbot
const chatbot = new AgenticChatbot({
    title: 'My AI Assistant',
    welcomeMessage: 'Hello! How can I help you today?',
    primaryColor: '#007bff',
    position: 'bottom-right'
});

chatbot.init();
</script>

Customize the chatbot to match your brand and requirements:

const chatbot = new AgenticChatbot({
    // Basic settings
    title: 'Your Assistant Name',
    welcomeMessage: 'Custom welcome message',
    primaryColor: '#your-brand-color',
    position: 'bottom-right', // or 'bottom-left', 'top-right', 'top-left'
    
    // Advanced settings
    apiKey: 'your-search-api-key', // For real web search
    maxResults: 5, // Number of search results to show
    autoOpen: false, // Don't auto-open on page load
    showTimestamp: true // Show message timestamps
});

üîç API Setup

API Key Sources

  • OpenAI API: Get your key from OpenAI Platform
  • Serper API: Get your key from serper.dev

üîß Extending the System

The modular architecture makes it easy to add new capabilities:

// Example: Adding a booking agent
class BookingAgent {
    async process(message) {
        // Handle booking requests
        return 'I can help you book tickets, appointments, etc.';
    }
}

// Register the new agent
chatbot.agentSystem.registerAgent('booking', new BookingAgent());
